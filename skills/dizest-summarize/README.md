# dizest-summarize

Summarize long-form content and turn it into structured, searchable knowledge.

Podcasts, articles, research papers, PDFs — one workflow. Powered by the API behind [Dizest: AI Summarizer](https://www.dizest.ai).

## What It Does

Pass any content to this skill — a URL, a YouTube podcast, a research paper, a PDF, or plain text — and get back a structured summary with key findings, takeaways, and action items.

- Research papers and academic content
- Long podcasts, interviews, and video content
- Articles, blog posts, and web pages
- PDFs, reports, and business documents
- Plain text, notes, and transcripts

Supports custom focus instructions (e.g., "focus on methodology") and output in any language.

## Setup

1. Get an API key at [dizest.ai/api/keys](http://dizest.ai/api/keys) (requires a paid Dizest account)
2. Set the environment variable:
   ```bash
   export DIZEST_API_KEY=your_api_key
   ```

## Usage

Once installed, invoke the skill by asking your agent to summarize content:

- "Summarize https://example.com/article"
- "Summarize this podcast https://youtube.com/watch?v=..."
- "Summarize this research paper and focus on the key findings"
