# AI Music Generator Prompts

## Formatting for Suno/Udio

### Section Tags (Required)
```
[Verse 1]
[Pre-Chorus]
[Chorus]
[Verse 2]
[Bridge]
[Outro]
```

### Optional Tags
```
[Intro]
[Instrumental]
[Solo]
[Break]
[Drop]
[Build]
```

### Performance Tags (in brackets)
```
[Verse 1 - half-whispered]
[Chorus - belted, emotional]
[Bridge - spoken word]
[Outro - fade out]
```

### Formula:
```
[Genre], [Era/Style], [Tempo], [Mood], [Vocal Type], [Instrumentation], [Production Notes]
```

## Instrumentation Tags

### Acoustic:
- Fingerpicked guitar, strummed acoustic
- Piano, grand piano, upright piano

- Horns, brass section

### Electronic:
- Synth pads, arpeggios, leads
- 808s, drum machine, live drums
- Bass synth, sub bass
- Ambient textures

### Production:
- Lo-fi, tape saturation, vinyl crackle
- Clean, polished, radio-ready

---

## Iteration Prompts

### After first generation:
- "More energy in the chorus"
- "Less busy verses, more space"
- "Brighter vocal tone"
- "Add pre-chorus build"
- "Slower tempo, more ballad feel"
- "Different chord voicings"
- "More pronounced hook"

### Negative prompts (what to avoid):
- "No autotune"
- "Avoid generic pop production"
- "No excessive reverb"
- "No vocal fry"
- "Avoid EDM drops"
