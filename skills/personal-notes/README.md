# Personal Notes

Персональный блокнот и дневник для OpenClaw. Записывает мысли, рефлексии, заметки с датой и временем.

## Что умеет

- Записывать мысли и заметки мгновенно
- Вести ежедневный дневник
- Помогать с рефлексией
- Искать по заметкам
- Показывать заметки за день/неделю/месяц
- Теги для организации

## Команды

```
[напиши мысль]             — записать с датой и временем
дневник                    — начать запись дня
заметки за сегодня         — показать записи за сегодня
заметки за неделю          — показать за неделю
найди заметку про [тема]   — поиск по заметкам
#важно [текст]             — записать с тегом важно
инструкция                 — показать все команды
```

## Установка

```bash
clawdhub install personal-notes
```

## Данные

Хранятся в `knowledge/personal/notes.md` в воркспейсе агента.
