## Quick Start

```bash

# Audit your RSS feeds

```

## Configuration

### LLM Classification (recommended)

Set one of these environment variables for high-quality classification:

```bash

export OPENAI_API_KEY="sk-..."          # Uses GPT-4o-mini
```

Without an API key, Feed Diet falls back to keyword-based classification â€” decent but not as nuanced.

## How It Works

1. **Fetch** â€” Pulls your submitted stories from HN API or items from RSS feeds
2. **Classify** â€” Batches of 25 items sent to LLM for categorization (or keyword fallback)
3. **Report** â€” Generates Markdown with ASCII charts, insights, and recommendations
4. **Digest** â€” (Optional) Filters items by your stated goals for a curated reading list
